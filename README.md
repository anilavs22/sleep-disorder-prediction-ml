# Sleep Disorder Prediction using Machine Learning

## Overview
This project predicts sleep disorders using machine learning algorithms based on healthcare and lifestyle data.

## Technologies Used
- Python
- pandas
- NumPy
- scikit-learn
- Matplotlib
- Seaborn

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Multiple ML models
- Hyperparameter tuning
- Cross-validation
- Model saving using Joblib

## Models Used
- Logistic Regression
- KNN
- SVM
- Random Forest

## Output
Predicts:
- Insomnia
- Sleep Apnea
- No Sleep Disorder
